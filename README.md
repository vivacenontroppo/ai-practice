# Playwright E2E Automation

Basic end-to-end automation setup using Playwright, TypeScript and class-based Page Object Model.

## Scripts

- `npm test` - run tests headlessly
- `npm run test:headed` - run tests in a headed browser
- `npm run test:ui` - open Playwright UI mode
- `npm run typecheck` - run TypeScript checks

## Structure

- `src/pages` - page objects
- `src/components` - reusable page components
- `tests` - Playwright specs
